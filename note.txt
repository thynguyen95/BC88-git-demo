Các bước chuẩn bị repo:
1. Bước 1: Tạo local repository: git init 
2. Bước 2: Tạo remote repository: tạo repo trên trang github 
3. Bước 3: kết nối local repo và remote repo với nhau: 
    git remote add origin "url của remote repo"


Đẩy các file lên git server (remote repo)
1. khai báo những file cần đẩy lên remote repo 
    git add "tên file"
    git add -A 
    git add .

2. commit: message phải có ý nghĩa đúng với nội dung các file được đẩy lên remote repo
    git commit -m "message commit"

3. push code lên remote repo 
    git push -u origin "tên nhánh" --- chỉ dùng cho lần đầu tiên push lên 1 nhánh
    git push                       --- từ lần 2 trở đi trên 1 nhánh 


Tham gia dự án với vai trò được mời 
Bước 1: chủ dự án mời người tham gia thông qua mail 
Bước 2: người tham gia kéo source code của dự án về 
    git clone "url git của dự án"
